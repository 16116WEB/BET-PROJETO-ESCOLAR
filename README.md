🎲 Simulador de Dados

Um simulador interativo de apostas com dados desenvolvido em HTML, CSS e JavaScript puro, com foco em praticar conceitos de lógica de programação.

🚀 Funcionalidades
🎯 Simulação automática de 10 rodadas
🎲 Animação de dado em tempo real
💰 Sistema de saldo com ganhos e perdas
🧠 Tipos de aposta:
Número exato
Par ou ímpar
📊 Histórico completo das rodadas
🏆 Relatório final com resultado do jogo
🔄 Reset do jogo
🧠 Conceitos aplicados

Este projeto foi desenvolvido aplicando conceitos fundamentais de programação:

✅ Funções
✅ Objetos (jogador)
✅ Vetores (historico)
✅ Estruturas condicionais (if/else)
✅ Laços de repetição (controle de rodadas)
✅ Manipulação do DOM
✅ Animações com CSS
🖥️ Tecnologias utilizadas
HTML5
CSS3 (com animações e variáveis)
JavaScript (Vanilla)
📂 Estrutura do projeto
📁 projeto
 └── index.html

Todo o projeto está em um único arquivo.

▶️ Como executar
Baixe o arquivo index.html
Abra no navegador

OU acesse a versão online (se hospedado)

🎮 Como jogar
Clique em "INICIAR JOGO"
O sistema executa automaticamente as rodadas
Acompanhe:
saldo
vitórias/derrotas
histórico
Veja o relatório final
📊 Regras do jogo
Saldo inicial: 100 moedas
Apostas aleatórias entre 1 e 10
Tipos de aposta:
🎯 Exato → ganha 5x
⚖️ Par/Ímpar → ganha 2x
O jogo termina quando:
completar 10 rodadas
ou saldo chegar a 0
